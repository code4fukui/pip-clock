# pip-clock
English README: [README.md](README.md)

ピクチャインピクチャ(PiP)機能を備えた時計。

## デモ
デモは [GitHub上のリポジトリ](https://code4fukui.github.io/pip-clock/) で利用できます。

## 機能
- 大きな数字表示で現在時刻を表示
- PiP ウィンドウでクロックを起動できる

## 使用方法
時計を使うには:

1. `index.html`ファイルをウェブブラウザで開きます。
2. "PiP START"ボタンをクリックしてPiPウィンドウで時計を起動します。
